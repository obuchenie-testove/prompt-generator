# Prompt Generator

Уеб приложение за генериране на учебни промпти и базов админ панел.

## Първи стъпки

1. Копирайте примерната конфигурация и попълнете достъпа до MySQL:
   ```bash
   cp prompt/config.php.example prompt/config.php
   ```
   Редактирайте `prompt/config.php` с реалните креденшъли.
2. Импортирайте `schema.sql` в избраната база данни (например чрез phpMyAdmin → Import).
3. Отворете приложението:
   - Генератор: `/prompt/index.php`
   - Админ панел: `/prompt/admin/login.php` (вход: `admin@local` / `admin123`)

## Структура

- `prompt/index.php` – клиентска форма за генериране на промпти.
- `prompt/admin/` – вход и начално табло за администрация.
- `prompt/db.php` – PDO helper функции за достъп до базата.
- `schema.sql` – MySQL схема и начални данни.
