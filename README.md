## Настройка на база данни

1. Създайте база данни в MySQL и импортнете схемата:
   ```bash
   mysql -u <user> -p <database> < schema.sql
   ```
2. Копирайте примерния конфигурационен файл и актуализирайте достъпа:
   ```bash
   cp config.php.example config.php
   ```
   Редактирайте `config.php`, така че да съдържа правилните хост, база данни, потребител и парола.
