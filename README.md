# Prompt Generator

Първоначален README за Codex тестове

Този проект служи за тестване и експерименти с Codex и генерация на промпти.

## Описание

Това е начална версия на README файла. Моля, допълнете с информация за структурата, употребата и целите на проекта, когато е възможно.

## Административен панел и база данни

1. Създайте MySQL база данни и изпълнете скрипта `schema.sql`, за да получите структурата и началните данни:
   ```bash
   mysql -u root -p < schema.sql
   ```
2. Копирайте `config.php.example` като `config.php` и попълнете достъпните креденшъли за вашата среда.
3. Отворете `/admin/login.php` и влезте с потребител `admin@local` и парола `admin123`.
4. Попълнете глобалните настройки, каталозите и/или редактирайте шаблоните според нуждите.
5. Отворете `/generator.php`, за да използвате каталозите, шаблоните и контекста от базата данни при генериране на промпт.
